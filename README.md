# Titanic Dataset - Exploratory Data Analysis (EDA)

## Task Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on the Titanic dataset to identify patterns, trends, relationships, and anomalies using statistical summaries and visualizations.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## Dataset

Titanic Dataset

## Analysis Performed

* Dataset overview using `info()` and `describe()`
* Missing value analysis
* Categorical variable analysis using `value_counts()`
* Histograms for Age and Fare distributions
* Boxplots for outlier detection
* Correlation heatmap
* Pairplot analysis
* Scatterplots and survival-based visualizations

## Key Findings

* More passengers died than survived.
* Male passengers outnumbered female passengers.
* Most passengers traveled in third class.
* The majority of passengers were between 20 and 40 years old.
* Fare distribution is highly right-skewed with several outliers.
* First-class passengers generally paid higher fares.
* Passenger class showed a significant relationship with survival.
* Female passengers had higher survival rates than male passengers.
* Fare had a positive correlation with survival, while passenger class had a negative correlation with survival.

## Conclusion

The analysis revealed that factors such as passenger class, fare paid, and gender significantly influenced survival outcomes on the Titanic. Various visualizations helped uncover patterns, relationships, and trends within the dataset.
